35 /*
36  * How to print strings
37  *
38  * In many cases, you can use Printf(). However, it takes time from when this function is
39  * executed until it is drawn on the screen. If you want to output to the screen instantly,
40  * you have to call Flush().
41  *
42  * ErrPrintf() is used when a serious error occurs in the system. Since this function does
43  * not alloc memory internally, there is a high possibility that it will be executed normally
44  * even if there is a problem in the system. In order to prevent Printf() from overwriting the
45  * screen while this function is drawing, DisablePrint() should be called beforehand.
46  */
